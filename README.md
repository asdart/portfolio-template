# Refined Portfolio Template

A modern, refined portfolio template for designers, built with Next.js 14, Tailwind CSS, and Framer Motion.

## Getting Started

1.  **Install dependencies:**

    ```bash
    npm install
    # or
    yarn install
    # or
    pnpm install
    ```

2.  **Run the development server:**

    ```bash
    npm run dev
    # or
    yarn dev
    # or
    pnpm dev
    ```

3.  Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Features

*   **Responsive Design:** Fully responsive layout for all devices.
*   **Animations:** Smooth entrance animations using Framer Motion.
*   **Modern Styling:** Clean, minimalist aesthetic using Tailwind CSS.
*   **Components:**
    *   Hero Section
    *   Project Gallery
    *   Services List
    *   Digital Goods Shop
    *   Blog Section
    *   Testimonials
    *   Pricing Tables
    *   FAQ Accordion
    *   Contact Form CTA

## Customization

*   **Colors:** Modified in `src/app/globals.css` or Tailwind classes.
*   **Content:** Data arrays (projects, services, etc.) are located at the top of each component file in `src/components/`.
*   **Fonts:** Uses `Inter` from `next/font/google` by default.

## Project Structure

*   `src/app/`: App router pages and layout.
*   `src/components/`: Reusable UI components and page sections.
*   `src/lib/`: Utility functions.
